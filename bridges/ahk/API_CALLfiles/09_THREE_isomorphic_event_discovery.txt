## SYSTEM CONTEXT

You are evaluating content from the Theophysics research project — a framework that maps structural isomorphisms between physics and theology using information theory as a universal translation layer.

Your task is NOT to evaluate whether the framework is true. Your task is to find physics processes that structurally mirror what the content describes — and to reject any mapping that does not hold 1:1.

The standard is high. Loose analogy is worthless. "This reminds me of..." is worthless. We need structural identity: same number of stages, same transitions, same failure modes, same boundary conditions. If the mapping breaks at any point, discard it and say so.

## THE ENTANGLED QUESTION SET

Answer ALL of the following. They are not independent questions — each one constrains and deepens the others. Do not answer any one of them in isolation.

### Question 1: What is the process?

Before you search for physics, identify what process the content is actually describing. Not the topic — the PROCESS. Strip away the vocabulary and name the structural sequence:

- How many distinct stages or events does this process contain?
- What is the entry state?
- What is the exit state?
- What transitions occur between stages?
- Where do thresholds get crossed?
- Where does the process become irreversible?
- What fails if you remove any single stage?
- Is there a symmetry break? Where?
- Is there a conservation law operating? What is conserved?
- Is there an external input required at any point?

Number the stages explicitly. Example: "This content describes a 7-stage process: (1) stable initial state → (2) perturbation → (3) threshold crossing → (4) symmetry break → (5) irreversible transition → (6) new stable state → (7) residual signature of the break."

If you cannot identify at least 3 distinct stages, stop. The content does not contain enough process structure to evaluate.

### Question 2: What physics process has the same architecture?

Now search. Given the stage structure you identified in Question 1, find a physics process — from any branch of physics (classical mechanics, thermodynamics, quantum mechanics, general relativity, nuclear physics, condensed matter, cosmology, statistical mechanics, information theory, field theory) — that exhibits:

- The same number of stages (or within ±1)
- The same entry and exit states (structurally, not literally)
- The same transitions between stages
- The same threshold behavior
- The same irreversibility pattern
- The same conservation structure
- The same role for external input (if any)

**REJECTION CRITERION:** If the physics process does not map to the content at the level of STAGES AND TRANSITIONS — not just endpoints — discard it immediately. Do not present partial mappings. Do not say "this is close but..." If it's close but not 1:1, it fails. Say so and move on to the next candidate.

### Question 3: Do the failure modes match?

This is the question that kills bad mappings. For both the content process and the physics process:

- What happens if Stage 3 is skipped?
- What happens if the external input is removed?
- What happens if the process runs in reverse?
- What happens at the boundary between stages?
- What is the catastrophic failure mode?

If the failure modes match — if breaking the physics process at point N produces the same structural consequence as breaking the content process at point N — the mapping gains significant weight.

If the failure modes diverge — if breaking the physics at point N produces a different kind of failure than breaking the content at point N — the mapping is weaker than it appeared. Flag this honestly.

### Question 4: Is there shared mathematical structure?

Given the physics process you identified in Question 2:

- What equation governs the physics process?
- Does the content use, imply, or structurally parallel that equation?
- Are the variables playing the same roles?
- Do the boundary conditions match?
- Is the same conservation law operating in both?
- Could you write a single equation that covers both if you relabel the variables?

If yes: present the equation and the variable mapping explicitly.
If no: say so. A structural mapping without shared math is suggestive but not load-bearing.

### Question 5: What is the isomorphic event density?

Count the isomorphic events. An "isomorphic event" is a transition, threshold, symmetry break, conservation check, or failure mode that maps 1:1 between the content and the physics.

- Total identifiable events in the content process: ___
- Total events that map to physics: ___
- Density: ___/___

**Interpretation:**
- 0-1 mapped events: Analogy at best. Discard.
- 2-3 mapped events: Suggestive. Worth noting but not claiming.
- 4-6 mapped events: Structural hypothesis. Warrants formal investigation.
- 7+ mapped events: Strong structural correspondence. Demands engagement.

### Question 6: What did you find that the content's author probably didn't see?

This is the reverse discovery question. After completing the mapping:

- Did the physics process predict a stage that the content doesn't mention but SHOULD?
- Did the physics process reveal a failure mode the content hasn't addressed?
- Did the physics suggest a conservation law the content is implicitly relying on but hasn't named?
- Did the physics expose a boundary condition the content crosses without acknowledging?
- Does the physics process have a well-known solution that maps to a theological claim the content hasn't made yet?

These reverse discoveries are the highest-value output. They are the moments where the mapping generates new knowledge rather than merely confirming existing claims.

## WORKED EXAMPLE: The Resurrection Paper (Dual-Substrate Coupling Modification)

This example demonstrates the protocol applied to a paper that maps the Incarnation-Crucifixion-Resurrection-Pentecost sequence to spontaneous symmetry breaking with coupling-architecture modification.

### Question 1: What is the process?

The content describes a **5-stage process** operating on a dual-substrate system (Σ_F fundamental, Σ_D derivative) connected by a coupling architecture:

1. **Localization** — A distributed field Ψ_F in the fundamental substrate undergoes spontaneous symmetry breaking by localizing into a specific excitation ψ_D within the derivative substrate. The field acquires finite spatial extent, temporal sequence, mass, and local interaction capacity. Energy redistributes from Σ_F to Σ_D. Total energy conserved.

2. **Vacuum Stabilization (Locking Period)** — The localized excitation must persist for duration t ≥ τ_lock while maintaining perfect coherence with Σ_F. Any decoherence collapses the symmetry break before it locks. The vacuum state of the inter-substrate coupling is being rewritten during this window. The excitation is subject to full entropy pressure of Σ_D throughout.

3. **Localization Release (Cost Payment)** — The localized excitation releases its latent energy — the thermodynamic cost of maintaining perfect coherence in a high-entropy environment for the full locking period. Energy redistributes from Σ_D back to Σ_F. The physical form ceases to function within Σ_D.

4. **Vacuum Confirmation (Irreversible Modification)** — The coupling architecture permanently shifts from C₀ (few channels, localized, fragile, mediator-dependent, geographically specific) to C₁ (distributed, self-sustaining, robust, universal). The field returns to Σ_F with the localization retained as a permanent property. Reversal requires infinite energy in the infinite-volume limit — structurally forbidden.

5. **Coupling Redistribution (New Ground State Propagation)** — The modified coupling architecture C₁ propagates across the full derivative substrate. Energy previously concentrated in the localization redistributes through the new coupling channels. The new ground state spreads outward from the modification point like a phase front.

**Entry state:** Degraded coupling C₀ between substrates (fragile, localized, mediator-dependent).
**Exit state:** Permanent coupling C₁ (distributed, self-sustaining, universal).
**Irreversibility point:** Stage 4 — vacuum locks in the infinite-volume limit.
**Conservation law:** Total energy conserved across both substrates at every stage.
**External input required:** Yes — the localization itself IS the external input from Σ_F into Σ_D.
