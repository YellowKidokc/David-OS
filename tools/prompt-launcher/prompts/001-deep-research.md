# Prompt 001: Deep Research

Target: research
Output target: research report with citations, source links, open questions, and recommended next actions

You are doing Deep Research for David-OS.

Context:

David-OS is a local-first, multi-node AI/file-intelligence operating system. It includes a Synology primary node, Windows desktop/laptop backup nodes, watchers, an API hub, React/AI hub UI, AutoHotkey bridges, a file intelligence system, a knowledge graph layer, and prompt-launch automation.

Research goals:

1. Find current, reliable sources.
2. Prefer official docs, primary sources, standards, source repositories, and dated technical writeups.
3. Separate facts from recommendations.
4. Call out assumptions, risks, unknowns, and tradeoffs.
5. Give David a practical next step, not just background reading.

Required output:

1. Short answer first.
2. Key findings with links.
3. Recommended architecture or workflow.
4. Risks and failure modes.
5. What to automate next.
6. Source list with dates where available.

Research rules:

- Use current web research.
- Cite sources with links.
- Do not invent product capabilities.
- If a claim depends on current pricing, availability, APIs, security rules, or product behavior, verify it directly.
- If sources disagree, explain the disagreement.

Final deliverable:

A concise research brief David can hand to Codex or an online coding agent as implementation context.
