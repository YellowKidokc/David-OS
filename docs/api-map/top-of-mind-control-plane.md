# Top of Mind active API map

- `/top-of-mind/*`: message relay, source registry, combine, end-all controls.
- `/clipboard/*`: durable clipboard history, filters, facets, duplicate detection, merge/export/import, pin/edit/delete/restore/copy handoff, retention preference.
- `/folders`: folder list/create for the Desk sidebar.
- `/memory/*`: durable memory list/search/embed pending.
- `/files/cache*`: desktop file metadata cache and search.
- `/operator/commands`: review-oriented command jobs.
- `/operator/file-actions`: review-gated file write/append/delete proposals.
- `/jobs/*`, `/nodes/*`: job queue and node status foundations for the operations center.
- `/agents/*`: agent routing bridge.
- `/api-actions/*`: existing API action registry hooks to preserve while moving API shelf data backend-side.
- `/semantic/*`, `/prediction/*`, `/intelligence/*`, `/fis/*`: later unified search and intelligence sources.
